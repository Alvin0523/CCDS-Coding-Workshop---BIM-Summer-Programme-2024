# Arithmetics


| Operator | Name | Example |
|----------|----------|----------|
| + | Addition | x+y |
| - | Subtraction | x-y |
| * | Multiplication | x*y |
| / | Division | x/y |
| == | Equals | x==y |
| != | Not Equal | x!=y |
| > | Greater Than | x>y |
| < | Less Than | x<y |
| >= | Greater Than or Equal to | x>=y |
| <= | Less Than or Equal to | x<=y |
